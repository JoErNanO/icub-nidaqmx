Detailed Description

The NIDAQmxTaskConfig is the configuration object for the NIDAQmxTask.

Description

The NIDAQmxTaskConfig is the configuration object for the NIDAQmxTask. It contains all those parameters which are needed to configure the NIDAQmx task before running it.

The configuration of a NIDAQmxTask object is a fairly cumbersome taks. This class was created to simplify the interface for the end-user while maintaining a most flexible functionality.

Tested OS

Linux, Windows
